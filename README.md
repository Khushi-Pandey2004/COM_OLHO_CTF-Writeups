# 🔐 CTF Writeups – COM OLHO Practice

This repository contains my **daily CTF practice writeups** across multiple cybersecurity domains.
The focus is on **understanding attack techniques, tools, and real-world problem-solving approaches**.

## 📌 Categories Covered

* 🧪 Forensics
* 🌐 OSINT (Open Source Intelligence)
* 🖼️ Steganography
* 🌍 Web Exploitation *(upcoming)*

## 🧠 Approach

For each challenge, I follow a structured methodology:

1. **Understanding the problem**
2. **Identifying possible attack vectors**
3. **Using appropriate tools**
4. **Documenting step-by-step solution**
5. **Extracting key learning**
